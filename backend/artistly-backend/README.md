# Artistly Platform Backend

This is a Spring Boot backend for the Artistly Platform using PostgreSQL as the database.

## Features
- Spring Boot REST API
- PostgreSQL integration
- JPA/Hibernate for ORM

## Getting Started

### Prerequisites
- Java 17 or later
- Maven
- PostgreSQL

### Configuration
Edit `src/main/resources/application.properties` with your PostgreSQL credentials.

### Build & Run
```
mvn clean install
mvn spring-boot:run
```

## Folder Structure
- `src/main/java` - Java source code
- `src/main/resources` - Configuration files
